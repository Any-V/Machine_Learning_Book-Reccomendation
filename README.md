# Machine_LearningA2_Book-Reccomendation
Machine Learning A2 Git repository of a machine learning model that reccomends books based on category and seniment analysis of book descriptions and reviews. 

## Project Overview
- _Text Classification_ used to predict book categories
- _Sentiment Analysis_ to understand user tone
- _Vector Search_ to find semantically similar books
- _Gradio UI_ for an easy-to-use and interactive reccomendation interface

| File | Description |
|------|-------------|
| `data-exploration.ipynb` | Initial EDA on book metadata and descriptions |
| `text-classification.ipynb` | Category prediction model |
| `sentiment-analysis.ipynb` | Sentiment scoring of reviews |
| `vector-search.ipynb` | Embedding generation + similarity search |
| `gradio-dashboard.py` | Interactive UI for recommendations |
| `tagged_description.txt` | Preprocessed text used for modelling |
| `cover-not-found.jpg` | Placeholder image for missing book covers |
